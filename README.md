## Sobre mim

Sou engenheira de software backend com experiência no desenvolvimento de sistemas críticos de alta disponibilidade, atuando principalmente com **C#/.NET** e **AWS**.

Hoje trabalho com soluções de **antifraude em larga escala**, onde atuo desde o desenvolvimento até decisões de arquitetura, com foco em **performance, escalabilidade, resiliência e custos (FinOps)**.

Tenho experiência com:
- Arquiteturas **serverless e event-driven**
- Modernização de sistemas **monolíticos para distribuídos**
- Integração entre serviços e processamento assíncrono
- Observabilidade e confiabilidade de sistemas
